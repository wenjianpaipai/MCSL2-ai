## 用到的开源项目

请查看 [此处](https://github.com/MCSLTeam/MCSL2/blob/master/pyproject.toml)  

## 从源码构建打包版

```bash
git clone https://github.com/MCSLTeam/MCSL2.git
uv sync

uv run deploy.py
或者添加 Nuitka 参数
uv run deploy.py --disable-console
```
